🔐 Secure Login System (Flask)

A simple and secure web-based authentication system built using **Python Flask** and **SQLite**. This project demonstrates core concepts of user authentication, password security, and session management.

---

## 📌 Features

- User Registration
- Secure Login System
- Password Hashing using **bcrypt**
- Session Management (Login/Logout)
- Protection against SQL Injection
- Basic Input Validation

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Database:** SQLite
- **Frontend:** HTML
- **Security:** Flask-Bcrypt
